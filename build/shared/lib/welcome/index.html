Or if you're familiar with 4.0, you can read about revisions and updates in recent 4.0 releases.

Processing 4.0 contains dozens of new features and hundreds of bug fixes. Among the new features:

  • Visit Tools → Theme Selector to choose a color theme.
  • Don't like sketch_220809a? Select an alternate sketch naming scheme in the Preferences window.
  • A new Movie Maker tool that can create animated GIFs—as well as high-resolution lossless MPEG videos of your work.
  • Apple Silicon is now supported (and very speedy).
  • Sketch and folder naming restrictions have been relaxed, making it easier to use version control (like Git) with your projects.
  • Many changes have been made to future-proof the software, like using Java 17.
  • We have versions of Processing for Raspberry Pi!
  • ...plus an absolutely massive number of small bug fixes and usability tweaks. Have fun!
